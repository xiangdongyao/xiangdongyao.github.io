GAMIT软件安装

发表于 2019-10-21 更新于 2020-02-01 分类于 GNSS - GAMIT

在Elementary OS 5 Juno是基于最新版本的Ubuntu，其平台上安装gamit无法成功，是gcc、gFortran版本太高的原因，解决方式如下：

出现错误

解决方式

删掉高版本的gcc/gfortran

在/usr/bin下

gamit安装成功

ubnutu最新的也适用
